# Al Amal Mortgage

Premium, lead-generation website for a UAE mortgage consultancy.

## Development

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

## Pages

- Home
- About
- Services
- Mortgage Calculator
- Eligibility Checker
- Contact

## Notes

- Placeholder imagery uses remote URLs and should be replaced with approved assets.
- Forms are static placeholders and should be connected to a CRM or API.
- Update contact details, WhatsApp number, and office information before launch.
